\doxysubsection{Detailed Description}
A class for managing Digital Elevation Model (DEM) data. 

The \doxylink{classMEMPA_1_1BUF__DEM}{BUF\+\_\+\+DEM} class is responsible for loading and preparing DEM data using the GDAL library. It handles a pointer to a GDALDataset and provides functions for accessing and processing elevation data.

This class includes\+:
\begin{DoxyItemize}
\item {\ttfamily dem\+\_\+fp} A pointer to a GDALDataset that represents the loaded DEM.
\item {\ttfamily out\+\_\+fp} A file path for the input DEM file.
\item {\ttfamily dem\+\_\+dataset} An output directory for saving processed data and results.
\end{DoxyItemize}

The class provides functionality to\+:
\begin{DoxyItemize}
\item Load a DEM file from the specified file path.

\begin{DoxyRefDesc}{Todo}
\item[\mbox{\hyperlink{todo__todo000002}{Todo}}]Retrieve metadata and information about the DEM. 

Divide the DEM into smaller, more useful chunks. 

Save processed outputs to the designated directory.\end{DoxyRefDesc}

\end{DoxyItemize}

This class is designed to facilitate easy integration of DEM data into larger geospatial processing workflows for the Mars Buffs CSCI senior project. (Fall 2024 -\/ Spring 2025)
