\doxysubsection{Detailed Description}
A class for managing Digital Elevation Model (\doxylink{classMEMPA_1_1DEM}{DEM}) data. 

The \doxylink{classMEMPA_1_1DEM}{DEM} class is responsible for loading and preparing \doxylink{classMEMPA_1_1DEM}{DEM} data using the GDAL library. It handles a pointer to a GDALDataset and provides functions for accessing and processing elevation data.

This class includes\+:
\begin{DoxyItemize}
\item {\ttfamily dem\+\_\+fp} A pointer to a GDALDataset that represents the loaded \doxylink{classMEMPA_1_1DEM}{DEM}.
\item {\ttfamily out\+\_\+fp} A file path for the input \doxylink{classMEMPA_1_1DEM}{DEM} file.
\item {\ttfamily dem\+\_\+dataset} An output directory for saving processed data and results.
\end{DoxyItemize}

The class provides functionality to\+:
\begin{DoxyItemize}
\item Load a \doxylink{classMEMPA_1_1DEM}{DEM} file from the specified file path.

\begin{DoxyRefDesc}{Todo}
\item[\mbox{\hyperlink{todo__todo000002}{Todo}}]Retrieve metadata and information about the \doxylink{classMEMPA_1_1DEM}{DEM}. 

Divide the \doxylink{classMEMPA_1_1DEM}{DEM} into smaller, more useful chunks. 

Save processed outputs to the designated directory.\end{DoxyRefDesc}

\end{DoxyItemize}

This class is designed to facilitate easy integration of \doxylink{classMEMPA_1_1DEM}{DEM} data into larger geospatial processing workflows for the Mars Buffs CSCI senior project. (Fall 2024 -\/ Spring 2025)
